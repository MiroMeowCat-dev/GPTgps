# GPTgps 快速使用（中文）

## 这是什么

GPTgps 是一个面向 ChatGPT 长对话场景的导航扩展。核心目标是：

- 快速定位历史提问/节点
- 对复杂任务按段管理
- 用标记点（Mark）做精确跳转
- 用 AI 自动生成段落与条目摘要

## 主要功能

- 分段（Segment）：从一整段对话中手动切割逻辑阶段
- 节点（Prompt / Mark）：
  - Prompt：用户提问节点
  - Mark：你手动放置的检查点，可跳到正文指定位置
- Pin：置顶重点节点
- Note：给每个节点写简短备注
- 搜索：按当前显示文本 + note + marker 标签搜索
- AI 总结：
  - 段级总结
  - 可选的 Prompt/Mark 级总结
  - 失败重试、错误提示、回退策略

## 安装方式（推荐）

### Chrome / Edge

1. 打开 `chrome://extensions`
2. 开启右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择项目目录中的 `dist_mv3`

### Firefox

1. 打开 `about:debugging#/runtime/this-firefox`
2. 点击“加载临时附加组件”
3. 选择 `dist_mv2/manifest.json`

## AI 配置说明

在 GPTgps 侧栏中打开 AI 设置面板，填写：

- Provider
- API Key
- Model
- Base URL

保存后会自动参与摘要生成。若失败，界面会显示失败原因与重试入口。
